# DIY PC Card

## overview

PC Engine homebrew.

4Mbit Flash memory card.

## files

pce_card1.comp : memory card's PCB CAD data(CADLUS X).

pce_wr.comp : writer's PCB CAD data(CADLUS X).

wr.prg : Writing software(Pi STARTER).
